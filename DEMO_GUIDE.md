# 计划书数据提取工具演示指南

## 🎯 快速演示

### 第一步：启动服务

#### 启动后端
```bash
cd /var/www/harry-insurance
python3 manage.py runserver 0.0.0.0:8007
```

#### 启动前端
```bash
cd /var/www/harry-insurance/frontend
npm run dev
```

### 第二步：访问应用
1. 打开浏览器
2. 访问前端URL（通常是 http://localhost:5173）
3. 点击导航到"计划书数据提取工具"

## 🎬 演示场景

### 场景1：正常提取流程
1. ✅ 上传保险计划书PDF
2. ✅ 观察5步进度展示
3. ✅ 查看提取的受保人信息
4. ✅ 查看保险产品和保费信息
5. ✅ 滚动查看年度价值表
6. ✅ 点击"确认并保存到系统"

### 场景2：数据编辑流程
1. ✅ 上传并提取数据
2. ✅ 点击"编辑数据"按钮
3. ✅ 修改受保人姓名为"李四"
4. ✅ 修改年龄为40
5. ✅ 点击"保存修改"
6. ✅ 确认数据已更新

### 场景3：错误处理演示
1. ❌ 尝试上传非PDF文件 → 显示错误提示
2. ❌ 上传超大文件 → 显示大小限制提示
3. ✅ 系统友好提示并阻止上传
