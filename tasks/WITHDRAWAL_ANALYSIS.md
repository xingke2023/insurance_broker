## 数据来源

- **基本计划**：md4.json - Projection 0（policyOptions: []）
- **提取计划**：md4.json - Projection 1（policyOptions: ['Withdrawal']）
- **投资假设**：当前假设情景（Current/Mid Scenario，非悲观/乐观情景）

## 重要说明

✅ **使用"当前假设情景"数据**
- 提取无Low/High后缀的列（如colAfterWDNotionalAmount）
- 代表中等/标准投资回报假设
- **不是"悲观情景"（Low）或"乐观情景"（High）数据**

**验证**：第80年名义金额 = 18,774（当前假设）vs 8,258（悲观情景）

## 关键发现

### 1. 复利效应的影响

提取款项会显著降低长期收益，因为：
- 每次提取减少了保单的投资基数
- 损失了被提取金额的复利增长机会
- 时间越长，差距越明显

### 2. 提取的机会成本

到第80年：
- 已提取现金：152,000
- 但总价值损失：3,504,347
- 每提取1元的机会成本：23.1元（在第80年）

### 3. 名义金额的变化

- 初始名义金额：50,000（估计）
- 第5年提取后：46,731（-3,269）
- 第80年提取后：18,774（-31,226）
- 名义金额随提取逐年递减

### 4. 适用场景

**适合提取计划的情况**：
- 需要稳定现金流
- 短期资金需求
- 年龄较大，不追求长期最大化

**适合基本计划的情况**：
- 追求长期财富最大化
- 不需要中途提取
- 作为遗产规划工具

## 数据提取说明

### 输出字段

```json
{
  "withdrawal": [
    {
      "policy_year": 80,              // 保单年度
      "premiums_paid": 50000,         // 累计已缴保费
      "withdrawal_amount": 2000,      // 该年提取款项
      "accumulated_withdrawal": 152000, // 累计提取款项
      "notional_amount": 18774,       // 提取后名义金额
      "guaranteed": 30493,            // 保证现金价值
      "non_guaranteed": 3135922,      // 非保证退保价值
      "total_surrender": 3166415      // 提取后退保价值
    }
  ]
}
```

### 使用程序

```bash
# 提取基本计划数据
python3 transform_json.py md4.json md41.json

# 提取款项提取计划数据
python3 transform_withdrawal.py md4.json md4_withdrawal.json
```
